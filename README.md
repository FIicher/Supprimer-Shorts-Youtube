<b>Installation :</b><br>
1- <i>Aller sur Google<br></i>
2- <i>Rechercher : extension Chrome<br></i>
3- <i>Une fois sur le site Chrome Web Store, installer : <a href="https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo">Tampermonkey</a><br></i>
4- <i>Activer l'extension<br></i>
5- <i>Aller sur le tableau de bord<br></i>
6- <i>Créer un nouveau script (appuyer sur l'icône " + ")<br></i>
7- <i>Copier entièrement le contenu du script ci-joint (antishorts.js)<br></i>
8- <i>Coller le contenu du script dans le tableau de bord tampermonkey, onglet: nouveau script<br></i>
9- <i>Sauvegarder, activer le script, redémarrer le navigateur<br><br></i>
