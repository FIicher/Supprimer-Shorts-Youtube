<b>Installation / Installation:</b><br>
1- <i>Aller sur Google / Go to Google<br></i>
2- <i>Rechercher : extension Chrome / Search for Chrome extension<br></i>
3- <i>Une fois sur le site Chrome Web Store, installer : <a href="https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo">Tampermonkey</a> / On Chrome Web Store, install: <a href="https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo">Tampermonkey</a><br></i>
4- <i>Activer l'extension / Enable the extension<br></i>
5- <i>Aller sur le tableau de bord / Go to the dashboard<br></i>
6- <i>Créer un nouveau script (appuyer sur l'icône " + ") / Create a new script (click the "+" icon)<br></i>
7- <i>Copier entièrement le contenu du script ci-joint (antishorts.js) / Copy the full content of the attached script (antishorts.js)<br></i>
8- <i>Coller le contenu du script dans le tableau de bord tampermonkey, onglet: nouveau script / Paste the script content in Tampermonkey dashboard, tab: new script<br></i>
9- <i>Sauvegarder, activer le script, redémarrer le navigateur / Save, enable the script, restart your browser<br><br></i>
